Додаткові інструменти

Для зручного та уніфікованого розміщення зображень у цьому тексті створено шість макросів. Їхню нотацію легко запам'ятати і використовувати:

Назва макросуДія макросу
image-center-caption(image, width)Центрований, з підписом,
без обтікання текстом
image-center(image, width)Центрований, без підпису,
без обтікання текстом
image-left-caption(image, width)Ліворуч, з підписом,
обтікання текстом праворуч
image-left(image, width)Ліворуч, без підпису,
обтікання текстом праворуч
image-right-caption(image, width)Праворуч, з підписом,
обтікання текстом зліва
image-right(image, width)Праворуч, без підпису,
обтікання текстом зліва

Зображення має зберігатися як внутрішній тідлер. У випадку використання зовнішнього зображення його зберігають як внутрішній тідлер із застосуванням механізму _canonical_uri. Підпис до зображення зберігається у його тідлері в полі caption.

Приклад використання:

<<image-left $:/languages/en-GB/icon 10%>>

Для використання цих макросів в інших ТВ потрібно скопіювати цей тідлер туди:

  • або шляхом перетягування посилання на цей тідлер (drag-and-drop),
  • або через експорт/імпорт тідлера (краще через формати .TID або .JSON).
